DTLS
-   Based on UDP
-   very secure, allows communication without the exchange being overheard or manipulated
-   fast and can reorder packets so they arrive in correct arrangement
-   good for gaming, watching online videos/streaming and video conferences/

SSL
-   Turns connection to https instead of regular http
-   Predecessor to TLS
-   Can generate self signed certificates using its own private key with command line
-   Can auto set the key and cetificate to the file names upon running


TLS
-   security protocol designed for privacy
-   mostly used for encrypting communication between web app and server. 
-   no impact to performance, except occasionly small load times for establishing connection
-   cloudfare offers free TLS/SSL certificates https://www.cloudflare.com/application-services/products/ssl/
